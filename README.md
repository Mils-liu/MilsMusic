# MilsMusic

![icon](https://upload-images.jianshu.io/upload_images/7019098-7939a38609d9c44e.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/60/format/webp)

## 实现功能

1.遍历本地音乐
2.音乐后台播放
3.音乐封面之黑礁唱片旋转效果
4.歌词的显示与切换（还未能实现平滑滚动效果）
5.摇摇切歌
6.本地天气信息的获取与显示
7.自定义音量控件
